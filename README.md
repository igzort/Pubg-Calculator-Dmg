# PUBG Damage Calculator v1.0

Удобный графический калькулятор урона для игры PUBG, написанный на Python. Позволяет визуально оценивать попадания по различным зонам тела персонажа с учетом выбранного уровня брони и шлема.

## 🎯 Функционал
* **Интерактивная хитбокс-карта**: Визуальный выбор зоны попадания на манекене персонажа.
* **Настройка снаряжения**: Удобный выбор уровней шлема (Helm L1-L3) и брони (Armor L1-L3).
* **Дистанция боя**: Изменение расстояния до цели кнопками с шагом в 10 метров.
* **База оружия**: Корректный расчет урона под конкретное выбранное оружие (например, M416).

## 💻 Установка и запуск из исходников

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Перейдите в папку проекта:
   ```bash
   cd PUBG_Calculator_dmg
   ```
3. Запустите главный скрипт:
   ```bash
   python pubg_calc.py
   ```

## 🛠️ Как собрать готовый .exe файл

Если вам нужно собрать переносимую версию программы для Windows со всеми спрайтами и встроенной иконкой сковородки, выполните команду в терминале:

```bash
pyinstaller --noconfirm --onedir --windowed --icon=скаравада.ico --add-data "hitbox_map.jpg;." --add-data "assets;assets" pubg_calc.py
```

После этого в корне проекта появится папка `dist/pubg_calc`, внутри которой будет находиться рабочий файл `pubg_calc.exe` и папка `_internal` со всеми ресурсами.

## 📂 Структура репозитория
* `pubg_calc.py` — главный файл инициализации приложения.
* `assets/` — папка со спрайтами элементов брони и интерфейса.
* `hitbox_map.jpg` — текстура манекена для выбора зон попадания.
* `скаравада.ico` — файл иконки приложения для сборки через PyInstaller.
